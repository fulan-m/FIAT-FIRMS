# Sobre

<div align='justify'>

Esse projeto foi desenvolvido com o apoio de uma bolsa de Iniciação Científica da Fundação de Amparo à Pesquisa do Estado de São Paulo (FAPESP) durante o período de 2025/2025. Este trabalho foi apresentado no Simpósio Mundial sobre Inteligência Artificial para Adaptação e Mitigação das Mudanças Climáticas (WS-AI4CCAM) na Universidade Federal Fluminense (UFF), Niterói - Rio de Janeiro, Brazil. Utilizando a linguagem _Python_, o _script_ utilizou o classificador Random Forest do pacote [Scikit-Learn](https://scikit-learn.org/) para treinar o modelo de predição de fogo utilizando as camadas de Focos de Incêndio / Anomalias de Temperatura (FIATs) dentro do bioma Cerrado contido no estado de São Paulo utilizando o produto NASA | FIRMS.

Os conjuntos de dados utilizados para todas as análises feitas no artigo não estão disponibilizados nesse repositório (limitações de armazenamento), mas os _scripts_ para gerá-los usando a API do Google Earth Engine (GEE) está disponível [aqui](https://github.com/fulan-m/FIAT-FIRMS/blob/main/articles/C.O.M.B.U.S.T.I.O.N./download_training_data.ipynb), para mais sobre o GEE [leia aqui](https://developers.google.com/earth-engine/tutorials/community/intro-to-python-api). As análises e modelagens estão agrupadas no script de _pipeline_ principal [aqui](https://github.com/fulan-m/FIAT-FIRMS/blob/main/articles/C.O.M.B.U.S.T.I.O.N./C_O_M_B_U_S_T_I_O_N-pipeline.ipynb)
